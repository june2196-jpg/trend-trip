About Prompt Atlas

사이트 소개

무엇을 제공하나요?

Prompt Atlas는 이미지 생성 AI와 챗GPT를 실무, 교육, 콘텐츠 제작에 활용하려는 사용자를 위한 프롬프트 라이브러리입니다. 단순 키워드 나열이 아니라 사용 목적, 적용 상황, 출력 형식을 함께 제안해 초보자도 바로 활용할 수 있도록 구성했습니다.

현재 제공하는 주요 콘텐츠는 이미지 생성 AI 프롬프트 100선, 실무 중심 AI 업무 비서 프롬프트, 회의록 정리와 이메일 작성 같은 업무 효율 프롬프트입니다. 앞으로는 실제 사용 예시, 수정 팁, 카테고리별 심화 가이드를 지속적으로 추가할 예정입니다.

콘텐츠 원칙

모든 콘텐츠는 사용자가 직접 응용할 수 있는 원문 설명과 예시를 중심으로 작성합니다. 과도한 자동 생성 복제 콘텐츠를 지양하고, 각 프롬프트가 어떤 상황에 적합한지 이해할 수 있도록 설명을 함께 제공합니다.

AI 도구의 결과는 항상 검토가 필요합니다. Prompt Atlas는 생산성 향상을 위한 참고 자료를 제공하며, 전문 자격이 필요한 조언이나 최종 의사결정을 대신하지 않습니다.

수집하는 정보

Prompt Atlas는 회원가입 기능을 제공하지 않으며, 사용자가 이름, 이메일, 전화번호 같은 개인정보를 직접 입력하는 기능을 운영하지 않습니다. 다만 Cloudflare Pages, Google AdSense, 웹 브라우저는 서비스 제공과 보안, 광고 표시를 위해 IP 주소, 브라우저 정보, 쿠키 정보를 처리할 수 있습니다.

Google AdSense는 맞춤 광고 제공과 광고 성과 측정을 위해 쿠키를 사용할 수 있습니다. 사용자는 브라우저 설정 또는 Google 광고 설정 페이지에서 맞춤 광고 사용을 관리할 수 있습니다.

이용 목적과 문의

수집될 수 있는 기술 정보는 사이트 접속 안정성, 보안, 통계, 광고 표시와 같은 목적에 사용됩니다. Prompt Atlas는 사용자의 개인정보를 별도로 판매하거나 임의로 제3자에게 제공하지 않습니다.

개인정보 또는 사이트 운영 관련 문의는 GitHub 저장소 이슈를 통해 접수할 수 있습니다. 운영자는 문의 내용을 확인한 뒤 필요한 범위에서 답변합니다.

이 페이지를 Cloudflare Pages에 바로 배포할 수 있나요?

네. 정적 파일만 사용하므로 저장소를 Cloudflare Pages에 연결하고 빌드 명령 없이 루트 디렉터리를 배포하면 됩니다.

AdSense 코드는 어디를 바꾸면 되나요?

현재 사이트에는 게시자 ID ca-pub-4898037749323009가 적용되어 있습니다. 광고 단위는 AdSense 심사 및 광고 단위 생성 후 추가하면 됩니다.

프롬프트를 더 추가하려면 어떻게 하나요?

main.js의 prompts 배열에 같은 형식의 항목을 추가하면 검색과 필터에 자동 반영됩니다.
