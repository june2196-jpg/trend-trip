# Prompt Atlas

이미지 생성 AI 프롬프트 100선을 공유하는 정적 HTML 페이지입니다. Cloudflare Pages에서 빌드 과정 없이 바로 서비스할 수 있습니다.

## Cloudflare Pages 배포

1. 이 저장소를 GitHub에 올립니다.
2. Cloudflare Pages에서 저장소를 연결합니다.
3. Build command는 비워둡니다.
4. Build output directory는 `/` 또는 프로젝트 루트로 설정합니다.

## AdSense 설정

`index.html`에는 아래 게시자 ID의 AdSense 자동 광고 스크립트가 적용되어 있습니다.

- `ca-pub-4898037749323009`

광고 단위별 수동 광고를 추가하려면 AdSense 심사 승인 후 실제 광고 단위 ID를 발급받아 `ins.adsbygoogle` 영역을 추가하세요. 현재 페이지에는 심사에 필요한 사이트 소개, 개인정보처리방침, 문의 섹션이 포함되어 있습니다.
